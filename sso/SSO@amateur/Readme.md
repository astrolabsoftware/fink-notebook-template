## Installation

- Step 1
```bash
conda create --name sso_env --file spec-file.txt
```

- Step 2
```bash
conda activate sso_env
```

- Step 3
```bash
pip install -r requirements.txt
```

- Step 4
```bash
python ssotuto_utils.py
```
